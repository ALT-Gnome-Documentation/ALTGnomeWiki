# Alacarte

Alacarte — это редактор меню для GNOME, использующий спецификацию меню freedesktop.org.

## Установка из репозитория

**Alacarte** можно установить через терминал:

::: code-group

```shell[apt-get]
su -
apt-get update
apt-get install alacarte
```         
```shell[epm]
epm -i alacarte
```
:::
